Despre proiect

Aplicația web permite înregistrare, autentificare JWT și gestionarea sarcinilor personale (creare, editare, finalizare, ștergere, filtrare). Interfața folosește Bootstrap 5 pentru un aspect consistent între pagina principală și această pagină.

Tehnologii

  • Backend: Python, FastAPI, Uvicorn
  • Bază de date: SQLite
  • Autentificare: JWT + bcrypt
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Deploy: Render.com (monorepo TWAAOS)
